html styiling

  • External CSS
    • To style html we use <link>Tag inside head tag
    • We have to make another file with .cssextension.
    • and add that file path inside link tag with the help of href="";attribute

    • example

      <link rel="stylesheet" href="html.css">
  • Internal CSS
    • To style html we use <style>Tag inside head tag
  • Inline CSS
    • To style html we use style="";attribute

For example

<h1 style="color:black";
